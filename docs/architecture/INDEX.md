# Architecture Index

## 说明

这个目录用于存放项目的架构文档。

- 按版本放在 `docs/architecture/<version>/` 下
- 当前最新版本目录是 `docs/architecture/v0.0.2/`
- 历史起始版本目录是 `docs/architecture/v0.0.1/`
- 同一版本内的小改动，直接更新对应版本文档
- 架构出现阶段性变化时，新增一个版本目录
